# cloudrail-si-node-sdk
Unified API Library for: Cloud Storage, Social Profiles, Payment, Email, SMS &amp; POIs. Included services are Dropbox, Google Drive, OneDrive, Box, Facebook, GitHub, Google+, LinkedIn, Instagram, Slack, Twitter, Windows Live, Yahoo, PayPal, Stripe, Mailjet, Sendgrid, Twilio, Nexmo, Google Places, Foursquare, Yelp.
